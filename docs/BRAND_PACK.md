# Modern Bank Brand Pack

## Identity
- Brand name: `Modern Bank`
- Visual program: `Asteria Private Series`
- Positioning: premium digital banking with institutional trust cues

## Core Palette
- Navy: `#0A2D56`
- Primary Blue: `#0D64C9`
- Cyan Accent: `#0F9F92`
- Trust Gold: `#B9852E`
- Mist Surface: `#E9F2FF`
- Primary Text Ink: `#132F52`

## Typography
- Heading family: `Sora`
- UI/body family: `Public Sans`
- Voice: concise, high-confidence, enterprise style

## Shared UI Components
- `brand-ribbon`: top gradient identity line for major views
- `brand-emblem`: reusable mark container for wordmark placements
- `brand-chip`: status/date/location metadata tags
- `primary-pill`: trusted-session badge with status dot
- `bank-seal`: institutional charter/info block

## Assets
- Mark icon: `/assets/modernbank-mark.svg`
- Favicon: `/assets/favicon.svg`

## Integration Rules
- Load `/css/style.css` first, then `/css/brand.css` for brand overrides.
- Use favicon on every page: `/assets/favicon.svg`.
- Use `brand.js` for dynamic date/time brand chips on dashboard/profile.
